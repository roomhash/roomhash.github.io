# RoomHash 共享市场

这是一个宿主无关、响应式的纯 Rust WASM 信息交易市场。公开目录、搜索、商品
详情、照片/视频描述符、发布和编辑流程、购买意向、卖家收件箱、滚动、命中检测
及所有可见界面均由 WASM 管理和绘制。应用不依赖 RoomHash HTML、CSS 或专用
JavaScript，可运行在任何兼容 `portable-surface-v1` 的宿主中。

## 产品边界

- 卖家发布、更新或撤下商品，公开价格、介绍、媒体和卖家联系方式。
- 买家确认购买后填写联系方式、收货/交付方式和备注。
- 敏感购买信息在 Rust WASM 内加密，只能由目标卖家私钥解密。
- 卖家收件箱明确提示主动联系买家，双方在线下商讨交付。
- 不提供线上付款、钱包、托管、支付状态、退款或资金流程。
- 不同节点只展示各自已收集并验证的数据，网络分区时目录可能不同。

## 构建和检查

```sh
npm run check
```

检查覆盖 Rust fmt/Clippy/单元测试、真实无 imports WASM、Ed25519/X25519/
HKDF/ChaCha20-Poly1305、所有权与 LWW、echo 排重、快照、买家隐私、卖家解密、
320×480 到 1440×900 的响应式场景，以及应用内全屏和安全随机数能力。

`dist/` 包含 `market.wasm`、`roomhash.json`、协议、README 和许可证。manifest
使用 `portable-surface-v1`；宿主只提供 Canvas、输入、文件做种、媒体加载、
密码学随机字节、持久化和不透明 P2P 转发。

## 媒体与隐私

照片/视频由宿主通用文件能力做种后返回内容寻址描述符。Rust WASM 验证 MIME、
大小、SHA-256、magnet/webSeed 和数量限制；媒体字节不进入 Gossip JSON。

身份种子在 WASM 内域分离派生 Ed25519 签名密钥和 X25519 加密密钥。宿主提供
的安全随机字节仅作为动作熵，密钥派生、nonce、加密、签名和所有市场语义仍在
WASM 内。公开网络可以看到双方 Hash 和密文元数据，但看不到买家联系方式、
交付方式和备注明文。

详见 `PROTOCOL.md`。
